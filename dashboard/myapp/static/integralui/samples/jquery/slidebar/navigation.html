SlideBar / Navigation Buttons


In this sample you can show/hide navigation buttons in SlideBar widget. By clicking on check box on the right, you can change the built-in properties:

  • showButtons - determines whether navigation buttons are visible or hidden

To navigate among slides, click on buttons. The order of buttons corresponds with the order of slides.

Whenever a button is clicked, the animation is interrupted and stopped, and in order to continue with animation, you need to call startAnimation method. More information can be found in Animations sample
