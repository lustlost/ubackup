In this sample you can show/hide icons for tabs in TabStrip widget. By clicking on check box on the right, you can change the built-in properties:

  • showIcons - determines whether an icon is shown before tab title

There are two ways to apply an icon to a tab:

  • By using HTML to set the data-element attribute to icon, or
  • By using JavaScript to set the icon variable of tab to some HTML code. You can use <img>, <span> or any other tag to add an image. Usually a <span> tag is used with image painted as background. For example:

    tab.icon = "<span class='icons heart' data-element='icon'></span>";
