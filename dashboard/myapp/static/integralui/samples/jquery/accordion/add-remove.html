IntegralUI Studio for Web

Accordion / Add - Remove







In this sample you can create and/or modify a list of groups using several different methods:

  • addGroup - adds a new group
  • insertGroupAfter - adds a new group after specified group (in this sample the selected group)
  • insertGroupAt - adds a new group at specified position
  • insertGroupBefore - adds a new group before specified group (in this sample the selected group)
  • removeGroup - removes a group (in this sample the selected group)
  • removeGroupAt - removes a group at specified position
  • clearGroups - removes all groups
