IntegralUI TreeView is a high performance light-weight widget which alows you to create and modify hierarchical structures on the fly. You can quickly load your data locally, remotely, or on demand. It has multiple methods optimized to quickly update the tree structure and current view.

Using links in right panel, you can check out different features of TreeView widget.
