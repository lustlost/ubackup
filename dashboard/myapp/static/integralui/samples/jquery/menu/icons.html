In this sample you can show/hide icons for items in Menu widget. By clicking on check box on the right, you can change the built-in properties:

  • showIcons - determines whether an icon is shown before the menu item title

There are two ways to apply an icon to an item:

  • By using HTML to set the data-element attribute to icon, or
  • By using JavaScript to set the icon variable of item to some HTML code. You can use <img>, <span> or any other tag to add an image. Usually a <span> tag is used with image painted as background. For example:

    item.icon = "<span class='icons heart' data-element='icon'></span>";
