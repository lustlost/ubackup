Menu / Local Data Binding


In this sample we are using a local data to load the content of Menu. In order this to work, you need to match the data fields from local data source to the fields of item variable. You can use any names for your data, as long as you match those names with item variable fields, the Menu will be populated.

You can set in this way all item properties like: icon, hyperlink, text, type, custom content etc.

For more information check out the source code of this sample.
