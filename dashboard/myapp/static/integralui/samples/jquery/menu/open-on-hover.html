In this sample you can change how sub-menus are opened. The Menu widget has a property openOnHover, which can accept one of these two values:

  • false - sub-menus are opened when root menu item is clicked, default
  • true - sub-menus are opened whenever mouse cursor hovers over parent menu item

When menus are opened on click, the root sub-menu will remain open until a click is made on some menu item or outside of the menu.
