IntegralUI TabStrip is a native AngularJS directive that allows you to organize your content in tabs. Each tab can have a header and content panel. The appearance and layout of tabs is determined by customizable CSS themes and styles.

Using links in right panel, you can check out different features of TabStrip directive.
