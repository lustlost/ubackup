SlideBar / Navigation Buttons

{{slide.name}}


In this sample you can show/hide navigation buttons in SlideBar directive. By clicking on check box on the right, you can change the built-in properties:

  • showNavigationPanel - determines whether navigation buttons are visible or hidden

To navigate among slides, click on buttons. The order of buttons corresponds with the order of slides.

Whenever a button is clicked, the animation is interrupted.
