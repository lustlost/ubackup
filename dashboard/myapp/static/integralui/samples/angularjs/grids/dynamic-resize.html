Grids / Dynamic Resize


This sample shows how columns in TreeGrid can occupy the whole space. Additionally, the TreeGrid will resize whenever a window size changes.

By default columns can have its width set in pixels, but in case of this example their width is set in percentage based on the TreeGrid width. So whenever TreeGrid is resized, the columns will also resize proportionally.

The following directive is used:

  • IntegralUI Resize - allows resizing of ANY element based on its parent, page or window. The tag for this directive is iui-resize.

The resize directive accepts an object which holds specification on how the corresponding element will resize. Both element width and height values can be set, using as a reference their 'parent', the 'page' (inner window area), or the 'window' (outer window area including the window frame).
